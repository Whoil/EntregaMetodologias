# METODOLOGÍA PL1

En este proyecto hemos implementado las clases de java correspondientes a PL1 de Metodología de la Progrmación.

El proyecto está organizado en:

- src/main/java: contiene las clases implementadas (Animal, Dog, Cat, Person, Student, Staff, etc.)
- src/test/java: contiene los tests de cada clase, utilizando JUnit


Los tests verifican:
- Getters y setters
- Métodos toString()
- Overrides
- Overloadings
- Polimorfismos

En el caso de métodos void, se comprueba que se ejecutan sin lanzar errores.
